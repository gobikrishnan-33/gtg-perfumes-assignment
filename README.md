🌸 GTG Perfumes — E-Commerce UI
A fully functional, multi-page perfume e-commerce website built from scratch using HTML5, CSS3, and Vanilla JavaScript — no frameworks. Developed as a frontend interview assignment to demonstrate real-world UI development skills including product filtering, cart management, and multi-page navigation.

🔗 Live Demo: https://gobikrishnan-33.github.io/gtg-perfumes-assignment/

🎯 Objective
Build a complete e-commerce frontend for a perfume brand — translating a design brief into a polished, interactive, multi-page shopping experience using only core web technologies.

🛠️ Tech Stack
TechnologyPurposeHTML5Semantic page structure across multiple pagesCSS3Responsive layout, product cards, animationsVanilla JavaScriptFiltering logic, cart state, DOM interactions

✨ Features

🛍️ Product Listing Page — Grid display of perfume products with clean card UI
🔍 Product Filtering — Filter products by category, price, or type dynamically
🛒 Shopping Cart — Add/remove items, quantity management, cart total calculation
📄 Multiple Pages — Separate pages with consistent navigation (Home, Products, Cart, etc.)
📱 Responsive Design — Optimized for desktop and mobile viewports
🎨 Brand-consistent UI — Clean, elegant styling matching a luxury perfume aesthetic

💡 Key Takeaways / What I Learned

Building a multi-page site architecture with shared navigation and consistent state
Implementing dynamic filtering logic in pure JavaScript without a library
Managing cart state (add, remove, update quantity, calculate total) using the DOM and JS objects
Writing reusable component patterns (product cards, nav) without a framework
Applying CSS for e-commerce layouts — grid-based product listings, sticky nav, responsive breakpoints
